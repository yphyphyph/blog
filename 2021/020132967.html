JavaWeb
发布日期:   2021-02-01

会话管理

HTTP协议是无状态（）

同一个浏览器的请求发送就认为是同一个会话，浏览器关闭本次会话结束

1.1. Cookie

1.1.1.简介

  • cookie由服务端产生，存储在浏览器端的硬盘中，当浏览器再次发请求给服务器时会带上所有的cookie
  • 默认情况下，当浏览器关闭会话结束的时候cookie过期；可以手动设置超时时间
  • 由于是存储在浏览器端，所以安全性较低，一般把不太重要的信息存储在cookie中
  • 浏览器存储cookie的大小是有限制的，每个服务器最多产生20个cookie

1.1.2.使用

1，服务器生成cookie

@WebServlet("/coo1")
public class CookieServlet extends HttpServlet &#123;
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException &#123;
        Cookie cookie = new Cookie("user", "0");
        Cookie cookie1 = new Cookie("pass", "0");

        //设置过期时间，时间单位是秒,10分钟后cookie过期
        cookie.setMaxAge(60 * 10);
        cookie1.setMaxAge(600);

        resp.addCookie(cookie);
        resp.addCookie(cookie1);
    &#125;
&#125;

2，再次请求时浏览器会带上cookie

@WebServlet("/coo2")
public class CookieServlet2 extends HttpServlet &#123;
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException &#123;
        //获取cookie
        Cookie[] cookies = req.getCookies();
        for (Cookie cookie : cookies) &#123;
            System.out.println(cookie.getName() + " : " + cookie.getValue());
        &#125;
    &#125;
&#125;

1.2. session

1.2.1. session简介

  • session是存储在服务器端 ，安全级别高
  • 默认情况下，浏览器关闭会话结束；session默认过期时间是30分钟

域对象

​ request：每次请求都会生成一个对象

​ session：同一个会话只会有一个session对象，同一个会话共用一个session对象

​ servletcontext：上下文对象，整个应用期间只有一个对象

1.2.2. session使用

1.获取session

@WebServlet("/sess1")
public class SessionServlet1 extends HttpServlet &#123;
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException &#123;
        // true如果没有session对象会新建， false表示不会新建session对象
        HttpSession session = req.getSession(false);
        System.out.println(session);

        session.setAttribute("user","1");
        //设置session的超时时间，2次请求的间隔时间超过指定时间会失效，时间单位是秒
        session.setMaxInactiveInterval(3);
//        System.out.println(session);
        session.setAttribute("","");
        System.out.println(session.getId());
    &#125;
&#125;

2.案例，记录用户登录状态

​ 1.首页

<body>
<a href="/shoppingCar">购物车</a>
<a href="/pay">结算</a>
</body>

​ 2.购物后台

@WebServlet("/shoppingCar")
public class ShoppingServlet extends HttpServlet &#123;
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException &#123;
        //1,验证用户是否登录过
        HttpSession session = req.getSession();
        Object user = session.getAttribute("user");
        if (user == null) &#123;
            //尚未登录
            resp.sendRedirect("/login.html");
        &#125; else &#123;
            //2，跳转到购物粗页面
            resp.sendRedirect("/shoppingcar.html");
        &#125;
    &#125;
