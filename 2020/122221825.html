类与对象

Java这门语言，是一门面向对象的语言；

C++ C# Java Java是纯面向对象语言；

程序设计思想：

  • 面向过程的设计思想 ，C
  • 面向对象的设计思想；涉及到那些对象 对象和对象之间有没有关联 合适的方法在合适的类里面；其实是对现实问题的抽象 跟容易让人理解

面向对象三大特征：

​ 封装 继承 多态

1.1.类与对象

  • 类是一个抽象的概念，是一组对象的蓝图，是把一组对象相同的特征（属性）和功能（方法)抽取出来
  • 类的组成有 属性 和方法
  • 对象是类的一个具体的实例（看得见摸得着的具体的实例）

1.1.1.如何定义一个类

1.属性（或者称为 实例变量/成员变量）

  • 成员变量的类型可以是任何数据类型
  • 成员变量有默认值，是其数据类型的0值，引用类型默认值是null
  • 成员变量的作用域是整个类体
  • 每个对象的成员变量有不同的存储空间

2.方法是类具有的功能；该类的所有对象共享该方法

1.1.3.成员变量和局部变量的区别

  • 声明的位置不同，局部变量在方法体种声明，成员变量在类体种声明

  • 作用域不同，局部变量的作用域是方法体，成员变量的作用域是整个类体

  • 局部变量使用之前一定要赋值才能使用，成员变量有默认值（JVM给的）数据类型默认值

  • 当成员变量和局部变量重名 的时候，优先使用局部变量

1.1.4.构造方法（构造器）

  • 用来完成对象的初始化动作，给成员变量赋了初始值
  • 构造方法的方法名和类名一样，没有返回值
  • 如果没有添加构造方法，会有一个默认的无参构造方法
  • 如果自己写了构造方法，不会再有默认的构造方法了
public class Student &#123;
